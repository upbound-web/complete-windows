# Website Editing Guide

## How to Edit Your Website (For Non-Technical Users)

Your Complete Window Cleaning website now has a **simple editing system** that lets you update text and images without touching any code!

---

## Getting Started

### Step 1: Open Your Website
1. Open `index.html` in your web browser
2. Look for the **"Edit Mode: OFF"** button in the top navigation bar

### Step 2: Turn On Edit Mode
1. Click the **"Edit Mode: OFF"** button
2. It will turn green and say **"Edit Mode: ON"**
3. A control panel will appear in the bottom-right corner

---

## Editing Text

When Edit Mode is ON:

1. **All editable text will have a green dotted outline**
2. **Simply click any text** (headings, paragraphs, bullet points, etc.)
3. **Type your changes** directly
4. **Click outside** the text box when done

### Examples of What You Can Edit:
- Headings (like "Professional Window Cleaning in Manning Valley")
- Descriptions and paragraphs
- Bullet point lists
- Customer reviews
- Prices
- Contact information

---

## Changing Images

When Edit Mode is ON:

1. **Each image will have a blue dotted outline**
2. **A "üì∑ Change Image" button** appears on each image
3. **Click the button** on the image you want to change
4. **Enter the new image URL or path** in the popup box
   - Example: `https://example.com/my-new-image.jpg`
   - Or: `public/my-new-photo.png`
5. **Click OK** to apply

---

## Saving Your Changes

### Option 1: Save to Browser (Quick Save)
1. Click the **"üíæ Save Changes"** button in the control panel
2. Your changes are saved in your browser
3. They will **automatically reload** next time you open the page
4. **Note:** Only works on the same computer/browser

### Option 2: Download Backup (Recommended)
1. Click the **"üì• Download Backup"** button
2. A file called `website-changes-YYYY-MM-DD.json` will download
3. **Keep this file safe!** You can reload it anytime
4. This lets you move changes between computers

### Option 3: Load a Backup
1. Click the **"üì§ Load Backup"** button
2. Select a previously downloaded backup file
3. All changes from that backup will be restored

### ‚úÖ DO:

## Troubleshooting

### "My changes disappeared!"
- Did you click "üíæ Save Changes" before closing the browser?
- If not, you'll need to re-do the changes
- **Always save before closing!**

### "I made a mistake and want to undo"
- Click "Edit Mode: OFF" then refresh the page (F5 or Cmd+R)
- This will reload your last saved version
- Or load a previous backup file

### "The image won't change"
- Make sure the image URL is correct and accessible
- Try using a direct image link (ending in .jpg, .png, etc.)
- Check that the image isn't too large (under 2MB recommended)

### "I accidentally deleted important text"
- Turn off Edit Mode and refresh the page
- Load your most recent backup
- If you don't have a backup, contact your developer

## Making Changes Live

### For Browser-Only Testing (No Server)
Your changes are saved locally in your browser. To share them:
1. Download a backup file
2. Send it to someone else
3. They can load it on their computer

### For a Real Website (With Server)
After making changes, you need to:
1. Download your backup file
2. Contact your web developer
3. They will apply the changes to the live website

**OR** if you're using GitHub Pages or similar:
1. Your developer can set up automatic saving to the server
2. Then your changes will be live immediately
